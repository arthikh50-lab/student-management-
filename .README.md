# student-api (Express + Mongoose)

## Endpoints
- `GET /health`
- `GET /api/students` (supports `search`, `page`, `limit`)
- `POST /api/students`
- `GET /api/students/:id`
- `PUT /api/students/:id`
- `DELETE /api/students/:id`

## Environment
Create `.env` based on `.env.example`.

Required:
- `MONGODB_URI`
- `PORT` (optional)

## Run
### Install
```bash
npm install
```

### Start
```bash
npm start
```

## Dev
```bash
npm run dev
```
